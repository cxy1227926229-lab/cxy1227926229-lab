### 2️⃣ 修改管理员密码

**位置：** `src\App.tsx` 文件，第69行

**步骤：**

1. 打开 `src\App.tsx` 文件
2. 按 `Ctrl + G` 跳转到第69行（或者按 `Ctrl + F` 搜索 `15351`）
3. 找到这一行：

```typescript
if (loginPassword === '15351') {
```

4. 将 `'15351'` 改成您想要的密码，例如：

```typescript
// 修改前
if (loginPassword === '15351') {

// 修改后（改成新密码：mypassword123）
if (loginPassword === 'mypassword123') {
```

5. **重要提示：**
   - 密码要用单引号 `'` 或双引号 `"` 包起来
   - 密码区分大小写（`ABC` 和 `abc` 是不同的）
   - 建议使用复杂密码，不要用简单的数字

6. 保存文件

### 4️⃣ 修改业务预设（下拉菜单选项）

**位置：** `src\core.ts` 文件，第9-11行

**步骤：**

1. 打开 `src\core.ts` 文件
2. 找到第9-11行：

```typescript
const PRESET_SERVICES = [
    '速写', '摄影', '300w盲盒', '500w盲盒'
```

3. 修改成您想要的业务名称，例如：


- 最后一个业务后面不要加逗号
- 每行一个业务，方便阅读

### 5️⃣ 修改颜色和样式

**位置：** `src\App.tsx` 文件，第18-36行

**步骤：**

1. 打开 `src\App.tsx` 文件
2. 找到第18-36行的 `ff14Theme` 部分：

```typescript
const ff14Theme = {
  background: 'linear-gradient(135deg, #0a0a0a 0%, #1a1a1a 50%, #0f0f0f 100%)',
  backgroundDark: '#0a0a0a',
  backgroundCard: 'linear-gradient(135deg, #1a1a1a 0%, #2a2a2a 100%)',
  gold: '#d4af37',
  goldLight: '#f4d03f',
  goldDark: '#b8860b',
  // ... 更多颜色
};
```

3. **修改颜色值：**

颜色使用十六进制代码，格式为 `#RRGGBB`，例如：
- `#d4af37` = 金色
- `#ff0000` = 红色
- `#00ff00` = 绿色
- `#0000ff` = 蓝色
- `#ffffff` = 白色
- `#000000` = 黑色

**常用颜色代码参考：**
- 金色：`#d4af37` 或 `#ffd700`
- 红色：`#ff0000` 或 `#dc3545`
- 蓝色：`#1f6feb` 或 `#007bff`
- 绿色：`#28a745` 或 `#00ff00`
- 紫色：`#6f42c1` 或 `#9c27b0`
- 粉色：`#e91e63` 或 `#ff69b4`

**示例：将金色改成蓝色**

```typescript
// 修改前
gold: '#d4af37',

// 修改后
gold: '#1f6feb',
```

**提示：** 可以在网上搜索"颜色代码表"或"十六进制颜色"找到更多颜色。

### 7️⃣ 修改公告生成模板

**位置：** `src\App.tsx` 文件，第82-88行

**步骤：**

1. 打开 `src\App.tsx` 文件
2. 找到 `announcementPreview` 部分（大约第82行）
3. 找到这一行：

```typescript
return `打扰致歉——请想要指定${target}速写业务的大人，在说话频道复制【/random】进行 roll 点，取点数${pickText}的${slotText}。`;
```

4. 修改成您想要的公告模板，例如：

```typescript
// 修改前
return `打扰致歉——请想要指定${target}速写业务的大人，在说话频道复制【/random】进行 roll 点，取点数${pickText}的${slotText}。`;
```

- 其他文字可以随意修改
- 使用反引号 `` ` `` 包起来（不是单引号）

## 🚀 修改后如何更新网站

### 重要提示

修改代码文件后，**必须重新构建和部署**，网站才会更新！

---

### 方法一：使用Vercel自动部署（推荐）

**如果您使用Vercel部署：**

1. **提交代码到GitHub**
   - 打开"命令提示符"或"PowerShell"
   - 输入以下命令（一行一行执行）：

```bash
cd C:\Users\Administrator\ffxiv-roll-tool
git add .
git commit -m "更新网站内容"
git push
```

2. **等待自动部署**
   - Vercel会自动检测到代码更新
   - 通常1-2分钟后，网站会自动更新
   - 可以在Vercel控制台查看部署状态

---

### 方法二：手动重新部署

**如果您使用Netlify或其他平台：**

1. **构建项目**
   - 打开"命令提示符"或"PowerShell"
   - 输入以下命令：

```bash
cd C:\Users\Administrator\ffxiv-roll-tool
npm run build
```

2. **等待构建完成**
   - 看到 "built in ..." 表示构建成功
   - 会生成 `dist` 文件夹

3. **重新部署**
   - **Netlify：** 将 `dist` 文件夹拖拽到Netlify部署区域
   - **其他平台：** 按照平台说明上传 `dist` 文件夹

---

### 方法三：使用GitHub Pages

1. **构建项目**
```bash
cd C:\Users\Administrator\ffxiv-roll-tool
npm run build
```

2. **部署到GitHub Pages**
```bash
npm run deploy
```

3. **等待几分钟后，访问您的网站查看更新**

## ⚠️ 修改注意事项

### 1. 语法错误

**常见错误：**
- 忘记加引号：`'文字'` 或 `"文字"`
- 忘记加逗号：多个项目之间要用 `,` 分隔
- 括号不匹配：`(` 和 `)` 要成对出现

**解决方法：**
- 如果网站无法运行，检查浏览器控制台（按 `F12`）查看错误信息
- 仔细检查修改的地方，确保语法正确

### 2. 备份文件

**修改前建议：**
- 复制一份原文件作为备份
- 或者使用Git提交代码，可以随时回退

### 3. 测试修改

**修改后建议：**
- 先在本地测试：运行 `npm run dev`，在浏览器查看效果
- 确认无误后再部署到线上

## ❓ 常见问题解答

### Q1: 修改后网站没有变化？

**A:** 可能的原因：
1. 没有保存文件（按 `Ctrl + S`）
2. 没有重新构建（运行 `npm run build`）
3. 没有重新部署
4. 浏览器缓存（按 `Ctrl + F5` 强制刷新）

### Q2: 修改后网站打不开了？

**A:** 可能是语法错误：
1. 检查是否有拼写错误
2. 检查引号、括号是否匹配
3. 查看浏览器控制台（按 `F12`）的错误信息
4. 恢复之前的代码，重新修改

### Q3: 如何撤销修改？

**A:** 方法：
1. 如果使用Git：`git checkout -- 文件名` 恢复文件
2. 如果有备份：复制备份文件覆盖原文件
3. 手动改回原来的内容

### Q4: 如何查看修改效果？

**A:** 本地测试：
1. 打开命令提示符
2. 输入：`cd C:\Users\Administrator\ffxiv-roll-tool`
3. 输入：`npm run dev`
4. 打开浏览器访问显示的地址（通常是 `http://localhost:5173`）

### Q5: 找不到要修改的文字？

**A:** 方法：
1. 使用 `Ctrl + F` 搜索功能
2. 如果找不到，可能是：
   - 文字在另一个文件中（检查 `core.ts`）
   - 文字是动态生成的（需要修改代码逻辑）

### Q6: 如何修改字体大小？

**A:** 在 `App.tsx` 中找到 `fontSize` 属性，例如：
```typescript
fontSize: 42,  // 改成您想要的大小，如 36、48 等
```

### Q7: 如何添加新的功能？

**A:** 这需要一定的编程基础，建议：
1. 学习基础的React和TypeScript
2. 参考现有代码的结构
3. 或者找有经验的朋友帮忙
