## 📝 修改步骤（3步）

### 1️⃣ 打开文件
- 用VS Code打开项目文件夹
- 找到 `src` 文件夹下的文件

### 2️⃣ 修改内容
- 按 `Ctrl + F` 搜索要修改的文字
- 直接修改
- 按 `Ctrl + S` 保存

### 3️⃣ 更新网站
```bash
npm run build
# 然后重新部署（Vercel会自动，其他平台手动上传dist文件夹）
```

## ⚠️ 注意事项

1. ✅ 修改后必须保存（`Ctrl + S`）
2. ✅ 必须重新构建（`npm run build`）
3. ✅ 必须重新部署
4. ✅ 清除浏览器缓存查看（`Ctrl + F5`）

## 🔧 常见问题

**Q: 修改后没变化？**  
A: 检查是否保存、构建、部署

**Q: 网站打不开？**  
A: 检查语法错误，查看浏览器控制台（`F12`）

**Q: 找不到文字？**  
A: 使用 `Ctrl + F` 搜索，可能在其他文件中
