## 📦 部署方式选择

### 方式一：Vercel（推荐，最简单）⭐

**优点：** 免费、自动部署、全球CDN、支持自定义域名

#### 步骤：

1. **注册Vercel账号**
   - 访问 https://vercel.com
   - 使用GitHub账号登录（推荐）或邮箱注册

2. **准备GitHub仓库**
   ```bash
   # 如果还没有Git仓库，执行以下命令：
   git init
   git add .
   git commit -m "Initial commit"
   
   # 在GitHub上创建新仓库，然后：
   git remote add origin https://github.com/你的用户名/ffxiv-roll-tool.git
   git branch -M main
   git push -u origin main
   ```

3. **在Vercel部署**
   - 登录Vercel后，点击 "Add New Project"
   - 选择你的GitHub仓库 `ffxiv-roll-tool`
   - **框架预设：** 选择 "Vite"
   - **构建命令：** `npm run build`（通常自动识别）
   - **输出目录：** `dist`（通常自动识别）
   - 点击 "Deploy"

4. **等待部署完成**
   - 通常1-2分钟即可完成
   - 部署完成后会得到一个类似 `https://ffxiv-roll-tool.vercel.app` 的网址

5. **访问网站**
   - 直接访问Vercel提供的网址即可使用
   - 可以点击 "Settings" → "Domains" 添加自定义域名

### 方式二：Netlify（同样简单）

**优点：** 免费、自动部署、支持表单、拖拽部署

#### 步骤：

1. **注册Netlify账号**
   - 访问 https://www.netlify.com
   - 使用GitHub账号登录

2. **部署方式A：通过GitHub（推荐）**
   - 点击 "Add new site" → "Import an existing project"
   - 选择你的GitHub仓库
   - **构建命令：** `npm run build`
   - **发布目录：** `dist`
   - 点击 "Deploy site"

3. **部署方式B：拖拽部署（最简单）**
   - 在项目根目录运行 `npm run build`
   - 将生成的 `dist` 文件夹拖拽到 Netlify 的部署区域
   - 立即获得网址

4. **访问网站**
   - 部署完成后会得到类似 `https://random-name-123456.netlify.app` 的网址

---

### 方式三：GitHub Pages（免费但需要配置）

**优点：** 完全免费、与GitHub集成

#### 步骤：

1. **安装GitHub Pages部署工具**
   ```bash
   npm install --save-dev gh-pages
   ```

2. **修改package.json**
   在 `scripts` 中添加：
   ```json
   "predeploy": "npm run build",
   "deploy": "gh-pages -d dist"
   ```

3. **修改vite.config.ts**
   ```typescript
   export default defineConfig({
     plugins: [react()],
     base: '/ffxiv-roll-tool/', // 改为你的仓库名
   });
   ```

4. **部署到GitHub Pages**
   ```bash
   npm run deploy
   ```

5. **启用GitHub Pages**
   - 在GitHub仓库页面，点击 "Settings"
   - 左侧菜单找到 "Pages"
   - Source选择 "gh-pages" 分支
   - 保存后访问：`https://你的用户名.github.io/ffxiv-roll-tool/`

### 方式四：Cloudflare Pages（免费且快速）

**优点：** 全球CDN、免费SSL、快速部署

#### 步骤：

1. **注册Cloudflare账号**
   - 访问 https://pages.cloudflare.com
   - 使用GitHub账号登录

2. **创建项目**
   - 点击 "Create a project"
   - 选择你的GitHub仓库
   - **框架预设：** Vite
   - **构建命令：** `npm run build`
   - **输出目录：** `dist`
   - 点击 "Save and Deploy"

3. **访问网站**
   - 部署完成后会得到类似 `https://ffxiv-roll-tool.pages.dev` 的网址

## ❓ 常见问题

### Q: 部署后页面空白？
A: 检查 `vite.config.ts` 中的 `base` 配置是否正确。

### Q: 如何更新网站？
A: 
- Vercel/Netlify/Cloudflare：推送代码到GitHub，自动重新部署
- GitHub Pages：运行 `npm run deploy`

### Q: 可以自定义域名吗？
A: 可以！在对应平台的设置中添加自定义域名即可。

### Q: 部署后数据会丢失吗？
A: 当前版本数据存储在浏览器本地，不同设备/浏览器之间不共享。建议添加后端存储。
