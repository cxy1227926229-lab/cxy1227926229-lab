# 手动推送代码到 GitHub

## 方法 1：使用命令行（推荐）

### 步骤：
1. 打开 PowerShell 或 Git Bash
2. 进入项目目录：
   ```bash
   cd C:\Users\Administrator\ffxiv-roll-tool
   ```

3. 检查状态（确认有未推送的提交）：
   ```bash
   git status
   ```

4. 推送到 GitHub：
   ```bash
   git push origin master
   ```

5. 如果提示需要身份验证：
   - 使用 Personal Access Token（推荐）
   - 或者使用 GitHub Desktop

---

## 方法 2：使用 GitHub Desktop（最简单）

### 步骤：
1. 下载并安装 GitHub Desktop：
   - 访问：https://desktop.github.com/
   - 安装后登录你的 GitHub 账号

2. 打开 GitHub Desktop，选择 "File" → "Add Local Repository"

3. 选择项目目录：`C:\Users\Administrator\ffxiv-roll-tool`

4. 在 GitHub Desktop 中，你会看到：
   - 左侧显示 "1 commit to push to origin/master"
   - 点击右上角的 "Push origin" 按钮

5. 等待推送完成

---

## 方法 3：使用 VS Code 的 Git 功能

### 步骤：
1. 在 VS Code 中打开项目

2. 点击左侧的源代码管理图标（或按 `Ctrl+Shift+G`）

3. 在源代码管理面板中，点击 "..." 菜单

4. 选择 "推送" 或 "Push"

5. 如果提示，选择 "origin" 和 "master"

---

## 方法 4：使用 Git 命令行（如果网络不稳定）

### 如果遇到网络问题，可以尝试：

1. **使用 SSH 代替 HTTPS**（如果已配置 SSH 密钥）：
   ```bash
   git remote set-url origin git@github.com:cxy1227926229-lab/cxy1227926229-lab.git
   git push origin master
   ```

2. **配置代理**（如果你使用代理）：
   ```bash
   git config --global http.proxy http://proxy.example.com:8080
   git config --global https.proxy https://proxy.example.com:8080
   ```

3. **增加超时时间**：
   ```bash
   git config --global http.postBuffer 524288000
   git config --global http.lowSpeedLimit 0
   git config --global http.lowSpeedTime 999999
   ```
